视窗 由子元素：标题、关闭按钮、内容面板 组合而成； 实现了IPopup接口,显示视窗请调用show(); 向视窗中添加内容时，使用addContent()方法；此方法会将参数作为显示对象添加到内容面板中。 author swellee
