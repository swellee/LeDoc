Classpublic class LMenu
InheritanceLMenu LList LBox LPane LContainer LComponent LSprite flash.display.Sprite
Implements IPopup

LMenu 控件创建可分别选择的选项的弹出菜单，弹出菜单可以具有所需的任何数目的子菜单级别。 菜单及子菜单的创建通过MenuItemVO数据集实现，具体请参考LeUI-book中的示例。 打开 Menu 控件后，此控件将一直可见，直到通过下列任一操作将其关闭： 1.调用 LMenu.hide() 方法。 2.用户点选已启用的菜单项。 3.用户在 Menu 控件外部单击。 author swellee
