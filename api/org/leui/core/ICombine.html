  contentPane : LContainer
[read-only] 内容面板，因为 ICombine 组件由多个组件拼合而成，虽然直接父类为LContainer，但如果作为普通容器使用时，通常将容器属性转嫁给自身的某个组合元素。 使用contentPane可得此组合元素，并向其中添加子对象。 注意，有些功能性ICombine的contentPane可能是null，因为这些复合组件拒绝接收子对象（如LRadioButton、LCombox、LScrollBar等)
